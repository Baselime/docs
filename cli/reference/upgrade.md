# upgrade

---

Upgrades the Baselime CLI to the latest version


```bash # :icon-terminal: terminal
baselime upgrade
```

### Examples

Command:

```bash # :icon-terminal: terminal
baselime upgrade
```

Output:

```txt # :icon-code: output
Downloading the Baselime CLI archive...
Successfully downloaded the baselime CLI, try 'baselime --help'

Upgraded to v0.0.5
```
